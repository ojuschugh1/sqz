pub fn start(self, transport: McpTransport) -> Result<()>

Start the server on the given transport.

For Stdio: reads JSON-RPC messages from stdin, writes responses to stdout. For Sse: starts a minimal HTTP server on the given port.

Source

pub fn watch_presets(&self) -> Result<RecommendedWatcher>

Wire preset hot-reload: watch preset_dir for TOML changes.

The watcher runs in a background thread and stores the new preset TOML in shared.pending_preset. It is applied on the next request.

Returns the watcher handle — drop it to stop watching.
