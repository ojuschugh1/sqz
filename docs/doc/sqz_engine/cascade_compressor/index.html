Module cascade_compressor 

Source
Expand description

Compression Cascades — multi-level compression that degrades gracefully as content ages out of relevance.

Level 0 (fresh): Full compressed content Level 1 (aging): Signatures + changed lines only Level 2 (old): File name + public API summary Level 3 (ancient): One-line reference with metadata

Each level is a lossy compression of the previous level, but sqz controls what’s lost (unlike the LLM’s compaction which is unpredictable).
