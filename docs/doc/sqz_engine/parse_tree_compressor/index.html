Module parse_tree_compressor

Expand description

Parse Tree Guided Code Compression — uses tree-sitter ASTs to identify low-entropy subtrees and collapse them to their signature line.

For each node in the parse tree, we compute the Shannon entropy of its text content (character-level). Nodes with entropy below the median are considered “low-entropy” (boilerplate, repetitive code) and are collapsed to just their first line (the signature). High-entropy nodes are preserved in full.
